# To-Do List App

A simple to-do list web application built using HTML, CSS, and JavaScript.

Users can:
- Add new tasks
- Mark tasks as completed
- Edit existing tasks
- Delete tasks from the list

This project was created while learning JavaScript and DOM manipulation.

## Features
- Add tasks dynamically
- Edit tasks
- Remove tasks
- Mark tasks as completed
- Interactive user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter a task
2. Click the "Add Task" button
3. Manage tasks using edit, complete, or delete options
